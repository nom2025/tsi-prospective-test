# 凡例位置設定方法

## 設定ファイル

`グラフ/legend_config.csv`ファイルで凡例の位置を設定できます。

## 設定項目

| 項目 | 説明 | デフォルト値 | 設定可能な値 |
|------|------|-------------|-------------|
| legend_loc | 凡例の基本位置 | `upper right` | `upper right`, `upper left`, `lower right`, `lower left`, `right`, `left`, `center`, `best` |
| bbox_x | 凡例のX座標（0.0〜1.0） | `0.95` | 0.0（左端）〜1.0（右端） |
| bbox_y | 凡例のY座標（0.0〜1.0） | `0.95` | 0.0（下端）〜1.0（上端） |

## 設定例

### 例1: 右上（デフォルト）
```csv
legend_loc,bbox_x,bbox_y
upper right,0.95,0.95
```

### 例2: 左上
```csv
legend_loc,bbox_x,bbox_y
upper left,0.05,0.95
```

### 例3: 右下
```csv
legend_loc,bbox_x,bbox_y
lower right,0.95,0.05
```

### 例4: 左下
```csv
legend_loc,bbox_x,bbox_y
lower left,0.05,0.05
```

### 例5: 中央上部
```csv
legend_loc,bbox_x,bbox_y
upper center,0.5,0.95
```

## 使用方法

1. `グラフ/legend_config.csv`を編集
2. `グラフ作成実行.bat`を実行
3. 凡例の位置が更新されます

## Excel形式での出力

`グラフ作成実行.bat`を実行すると、`グラフ/total_score_data.xlsx`が作成されます。
このExcelファイルでは：
- グラフデータを編集できます
- グラフの凡例位置をExcelで直接調整できます
- PowerPointにコピー&ペーストして編集できます

## PowerPointでの編集方法

1. Excelファイルを開く
2. グラフをコピー
3. PowerPointに貼り付け
4. PowerPointで凡例の位置を調整
